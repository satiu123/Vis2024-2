# ChinaVis2024 就业招聘市场数据可视化项目

## 项目简介

本项目是基于ChinaVis2024可视分析挑战赛的就业招聘市场数据集，使用D3.js和Echarts技术实现的交互式数据可视化分析系统。

## 数据集概况

- **数据规模**: 430,664条招聘通知
- **职位种类**: 169,540种
- **企业数量**: 267,296家
- **行业类别**: 158个
- **行政区划**: 371个

## 技术栈

### 核心技术
- **D3.js v7**: 地理可视化、网络图、树图等复杂可视化
- **Echarts v5**: 统计图表、散点图、雷达图
- **HTML5/CSS3**: 响应式布局
- **JavaScript ES6+**: 模块化开发

### 数据处理
- **Python 3.x**: 数据清洗与预处理
- **Pandas**: 数据分析
- **NumPy**: 数值计算

## 快速开始

### 1. 环境准备

```bash
# 克隆项目
git clone https://github.com/satiu123/Vis2024-2.git
cd ChinaVis2024

# 安装Python依赖（用于数据预处理）
cd scripts
pip install -r requirements.txt
```

### 2. 数据预处理

```bash
# 运行数据清洗脚本
python scripts/data_cleaning.py

# 运行数据聚合脚本
python scripts/data_aggregation.py
```

### 3. 启动可视化系统

```bash
# 使用Python启动HTTP服务器
python -m http.server 8000

# 或使用VS Code的Live Server插件
# 右键 index.html -> Open with Live Server
```

### 4. 访问系统

打开浏览器访问: `http://localhost:8000/src/index.html`

## 常见问题

### Q1: 数据加载失败？
A: 检查数据文件路径，确保已运行数据预处理脚本。

### Q2: 图表显示不完整？
A: 检查浏览器控制台错误信息，确认D3和Echarts库正确加载。

### Q3: 性能问题？
A: 确认使用了聚合后的数据，而非原始430k条数据。
